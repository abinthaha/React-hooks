# React Hooks

This is a sample project using React hooks
### Tech

Doesn't have so much packages as it a sample one:

* React
* React DOM
* Webpack

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd React-hooks
$ npm install
$ npm start / npm run start
```

For build...

```sh
$ npm run build
```
